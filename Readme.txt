DeCaesar -- атака на шифр простой замены методом частотного анализа.

Автор: Виктор Щербинин, victorshch@gmail.com

Для установки необходимы библиотеки Qt >= 4.5.0 (используется для работы с utf-8), boost_serialization, boost_program_options >= 1.40.0.
Перед установкой может понадобиться отредактировать файл config.pri, в котором указываются пути к хедерам и библиотекам boost'а.

Установка:

qmake
make

После установки исполняемый файл decaesar находится в bin. Там же лежат файлы со статистикой, так что перед запуском имеет смысл выполнить
cd bin

Опции запуска:
-m	интерактивный режим -- после подбора ключа позволяет пользователю "подправить" его. Без этой опции работает в автоматическом режиме (правильная расшифровка не гарантируется, хотя на обоих калибровочных текстах работало правильно)
-v	подробный режим -- вывод комментариев во время работы
-l {en, ru}	выбор языка, по умолчанию -- английский
-i {filename}	файл с зашифрованным текстом. Если не указан, то используется stdin
-o {filename}	файл, в который писать результат дешифровки в конце работы программы. По умолчанию stdout
-s {filename}	указание файла с частотами. Для английского по умолчанию используется en_stat.xml, для русского -- ru_stat.xml
-g	режим, при котором программа собирает частотную статистику по входному файлу, записывает ее в файл с частотами и выходит
-h	вывод справки

Примечания: 
1. Программа предполагает, что зашифрованы только символы алфавита, т. е. что пробелы и знаки препинания не зашифрованы.
2. При расшифровке английского текста может работать ~20 секунд, при расшифровке русского текста ~1 минуту. Чтобы не было скучно, можно использовать опцию -v.

При разработке программы использовалась статья [Thomas Jakobsen, A Fast Method for the Cryptoanalysis of Substitution Ciphers, 1995] (http://citeseerx.ist.psu.edu/viewdoc/summary?doi=10.1.1.55.89). Использована целевая функция, зависящая от частот биграмм и триграмм.
